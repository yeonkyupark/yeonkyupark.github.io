# 1. 빅데이터 개요

## 1.1. 빅데이터(Big Data) 정의
`빅데이터(Big Data)`란 기존 데이터베이스 관리 도구의 능력을 넘어서는 대량의 정형 또는 심지어 데이터베이스 형태가 아닌 비정형의 데이터 집합까지 포함한 데이터로부터 가치를 추출하고 결과를 분석하는 기술을 말한다.

## 1.2. 빅데이터 특징 
* 빅데이터 특징
  * Volume / 규모의 증가
  * Variety / 다양성
  * Velocity / 처리속도
  * Value / 가치
  * Veracity / 정확성
* 빅데이터 특징 분류
  * **3V: Volume, Variety, Velocity**
  * 5V: 3V + **Value, Veracity**
  * 7V : 5V + Validity, Volatility 
* 데이터 크기 단위 순서 
  * Byte < KB(Kilo) < MB(Mega) < GB(Giga) < TB(Tera) < PB(Peta) < EB(Exa) < ZB(Zetta) < YB(Yotta) < BB(Bronto) < GeB(Geop)

## 1.3. 빅데이터 유형
* 정형 데이터
* 반정형 데이터
* 비정형 데이터

## 1.4. 빅데이터 분석의 중요성

## 1.5. 빅데이터 요소 기술
* 빅데이터 요소기술: 빅데이터 수집, 저장, 공유, 처리, 분석 및 시각화 (한국정보화진흥원)
* ETL(Extract/Transform/Load): 데이터 추출, 변환, 적재

## 1.6. 빅데이터 분석 기획 능력
* NCS(국가직무능력표준, National Competency Standards) 기준
  * 도메인 이슈 도출하기
  * 분석목표 수립하기
  * 프로젝트 계획하기
  * 보유데이터 자산 확인하기
* 빅데이터 분석 직무
  * **대용량의** 데이터 집합으로부터 유용한 **정보를** 찾고 결과를 예측하기 위해 목적에 따라 **분석기술과** **방법론을** 기반으로 정형/비정형 데이터를 **구축**, **탐색**, **분석하고** **시각화를** 수행하는 업무

## 1.7. 빅데이터와 데이터베이스
* 데이터베이스(DB), 데이터베이스 관리시스템(DBMS)
* DIKW 피라미드
  * Data
  * Information
  * Knowledage
  * Wisdom

# 2. 분석목적 설정

## 2.1. 빅데이터 분석 및 활용
* 빅데이터 분석의 주요 목적의 6가지 카테고리 (Gartner그룹, The Big Data Value Model)
  * 고객분석
  * 제품 및 처리과정의 효율성 제고
  * 디지털 제품 및 서비스 제공
  * 운영 효율성
  * 디지털 마케팅
  * 리스크 관리 및 운영

## 2.2. 빅데이터 분석목표정의서
* 구성요소
  * 데이터 원천 파악: 데이터 정보 조사, 데이터 입수 난이도 조사
  * 분석접근 방안 및 적용 가능성 판단
  * 성과평가 기준: 정성 평가, 정량평가
* 작성 예
  * 기본정보, 데이터 정보, 데이터분석 적용성 판단, 성과측정 포함

# 3. 데이터 확보 및 분석 방안 설정

## 3.1. 데이터 확보 계획 수립
* 수립 절차: 목표 정의 -> 요구사항 도출 -> 예산안 수립 -> 계획 수립
* 데이터 확보 시 고려사항
  * 단계별로 분석 내용 문서화하여 WBS(Work Breakdonw Structure, 작업분할구조도)에서 각 단계의 주요 수행 내용을 작성
* WBS 작성 단계
  * 데이터 분석과제 정의
  * 데이터 준비 및 탐색
  * 분석 모델링 및 검증
  * 산출물 정리

## 3.2. 데이터 분석 방안 설정
* 빅데이터 분석 절차 (요구사항 분석, 모델링, 검증 및 테스트, 적용)
  * 문제인식
  * 관련 연구조사
  * 모형화
  * 자료수집
  * 자료분석
  * 분석결과 제시
* `데이터마이닝(Data Mininig)`
  * 대규모로 저장된 데이터 안에서 체계적이고 자동적으로 통계적 규칙이나 패턴을 찾아내는 기법
  * 데이터 속의 지식 발견(Knowledge Discovery in Database)라고도 함
  * 패턴인식, 기계학습, 인공지능 등 다양한 영역에서 활용
  * 주요 적용 사례
    * 분류: 의사결정 나무(Decision Tree)
    * 군집화: k-means Clustering(k평균군집화)
    * 연관성: 동시에 발생한 사건들 사이의 관계 정의
    * 연속성
    * 예측
